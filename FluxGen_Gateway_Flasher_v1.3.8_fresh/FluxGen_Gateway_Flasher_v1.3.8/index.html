Process

How to Flash

  1. Connect the ESP32 board to your computer with a USB data cable — not a charge-only cable.
  2. Click "Install Firmware" above. Your browser will ask you to select a serial port.
  3. Select the port that corresponds to your ESP32 — usually labelled CP2102 or CH340.
  4. Wait for the flashing to complete. This takes approximately 1–2 minutes.
  5. The device will automatically reboot when the process is complete.
After Flashing gateway ~ boot
01 Wait 10 seconds for the ESP32 to complete its first boot.
02 Connect to WiFi network: ModbusIoT-Config  /  password: config123
03 Open your browser and navigate to http://192.168.4.1 — no login required, direct access to the configuration page.

Recommended

Alternative: Command-Line Flasher

If the browser flash stalls or fails midway, use the included flash_firmware.bat script instead. It uses esptool — the same tool used by professional developers — and is more reliable for large firmware.

Quick Start esptool ~ flash
$ pip install esptool   (one-time setup, requires Python)
$ flash_firmware.bat   (Windows)  or  ./flash_firmware.sh   (Mac/Linux)
> Enter your COM port when prompted — that's it.

Support

Troubleshooting

  1. No ports shown? Install the USB-to-Serial driver for your board: CP2102 Driver  or  CH340 Driver
  2. Flash fails immediately? Hold the BOOT button on the ESP32 board while clicking Install, then release it once the process begins.
  3. Browser flash stalls at ~50%? This is a Web Serial limitation with large firmware. Use flash_firmware.bat instead — it's more reliable.
  4. Port busy? Close any serial monitors such as Arduino IDE, PuTTY, or VS Code Serial Monitor that may be holding the port open.
